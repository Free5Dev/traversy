# -----------------------BASIC COMMANDS-----------------
## Initialize local Git Repository
$git init 
## Add File(s) To Index
$git add 
## Checkout Status of Working Tree 
$git status
## Commit changes In index 
$git commit 
## Push To Remote Repository
$git push 
## Clone Repository Into A New Directory
$git clone
